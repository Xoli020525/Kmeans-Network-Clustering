This project demonstrates K-Means clustering applied to a dense network of nodes. 
The clustering is based on spatial coordinates, and results are visualized using 2D and 3D plots.
The clustering was performed with k = 7 and k = 3 to compare how the number of clusters affects grouping.

Technologies used:
- Python
- NumPy
- Pandas
- Matplotlib
- NetworkX
- Scikit-learn
